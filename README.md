# 🛍️ AI Smart Product Recommendation System using Deep Learning

## 📌 Project Overview

The **AI Smart Product Recommendation System** is a web-based
application that recommends products to users based on their purchase
history using a Deep Learning recommendation model. The system analyzes
historical sales data and provides personalized product suggestions to
improve the shopping experience.

## ✨ Features

-   Deep Learning-based recommendations
-   Personalized recommendations using User ID
-   Product image display
-   User purchase history
-   Trending products
-   Popular products this month
-   Recommendation explanation
-   Flask web application

## 🛠️ Technologies Used

-   Python
-   Flask
-   TensorFlow / Keras
-   Pandas, NumPy, Scikit-learn
-   HTML, CSS, Jinja2
-   Excel (.xlsx)
